# 图片文件夹

## 请将以下图片放到这个文件夹中：

### 1. payment-qrcode.png（或 .jpg）
你的收款二维码（微信或支付宝）

### 2. wechat-qrcode.png（或 .jpg）
你的微信个人二维码

## 获取方式：

**收款码：**
- 打开微信/支付宝
- 找到"收付款"
- 保存二维码图片

**微信二维码：**
- 打开微信
- 右上角"+" → "添加朋友" → "我的二维码"
- 保存图片

## 注意事项：

- 图片格式可以是 jpg、jpeg 或 png
- 如果使用其他格式，需要修改 result.html 中的文件名
- 建议图片大小不超过 500KB
- 二维码要清晰可扫描

## 示例文件结构：

```
images/
├── payment-qrcode.jpg  (你的收款码)
├── wechat-qrcode.jpg   (你的微信二维码)
└── README.md           (本文件)
```
